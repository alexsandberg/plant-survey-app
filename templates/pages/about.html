This application is a citizen science tool for surveying rare plants in Colorado. Utilizing a database of plants based on the CNHP Colorado Rare Plant Guide, anyone can make an account and add an observation of a plant, contributing data that can be utilized for conservation and scientific research.

The site and application are under development, with the following additional planned features coming soon:

• Full integration of the CNHP rare plant database
• Geospatial coordinate information for plant observations, with map visualizations
• Additional data fields and information for plants
• Plant database search functionality

Data is also available directly via an API. Click here to obtain an API key, and read the full documentation at https://github.com/alexsandberg/plant-survey-app/blob/master/README.md
